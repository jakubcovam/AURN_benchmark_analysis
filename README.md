# AURN Benchmark Analysis

This repository contains analysis code for the manuscript:

**Empirical predictability limits of urban air quality: 
Benchmark forecasts using simple statistical models**


Submitted to *Earth Interactions* (2026)

---

## Status

This repository is currently being prepared for publication.
Full code will be deposited here upon acceptance of the manuscript.

In the meantime, please contact the corresponding author at
michala.jakubcova@it.cas.cz with any questions.

---

## Contents (upon publication)

- Data preprocessing and quality control scripts
- Benchmark model implementations (persistence, climatology, MCR, GAM)
- Forecast evaluation metrics (R², RMSE, FSS)
- Meteorology normalization and trend analysis
- Figure reproduction scripts

---

## Data

Observational data are publicly available from the UK Automatic 
Urban and Rural Network (AURN):
https://get-air-pollution-data.service.gov.uk/
